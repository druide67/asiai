# asiai tui

Interaktives Terminal-Dashboard mit automatischer Aktualisierung.

## Verwendung

```bash
asiai tui
```

## Voraussetzungen

Erfordert das `tui`-Extra:

```bash
pip install asiai[tui]
```

Dies installiert [Textual](https://textual.textualize.io/) für die Terminal-Oberfläche.

## Funktionen

- Echtzeit-Systemmetriken (CPU, Speicher, Thermal)
- Engine-Status und geladene Modelle
- Automatische Aktualisierung mit konfigurierbarem Intervall
