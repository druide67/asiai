# asiai tui

Dashboard terminale interattiva con aggiornamento automatico.

## Uso

```bash
asiai tui
```

## Requisiti

Richiede l'extra `tui`:

```bash
pip install asiai[tui]
```

Questo installa [Textual](https://textual.textualize.io/) per l'interfaccia terminale.

## Funzionalità

- Metriche di sistema in tempo reale (CPU, memoria, termica)
- Stato dei motori e modelli caricati
- Aggiornamento automatico con intervallo configurabile
